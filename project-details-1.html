CRM - Customer Relationship Management

A study in building a functional client management application focusing on the automotive industry. Built on a Rails API backend, React frontend, utilizing a postgres database.

  • * For best experience, please view this app on a desktop *
  • Developed Javascript/React front-end and Rails/Active Record/Postgres back-end application
  • Modeled custom database schema and REST API with Active Record, Postgres, and Ruby on Rails
  • Leveraged asynchronous data retrieval from two 3rd party APIs to power text messaging and email retrieval.
